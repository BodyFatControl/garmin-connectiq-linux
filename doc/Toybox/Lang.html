Module: Toybox::Lang

Overview

The Lang module contains Monkey C language basic types.

  • Number - 32 bit signed integers

  • Float - 32 bit floating point numbers

  • Long - 64 bit integers

  • Double - 64 bit floating point numbers

  • Booleans - true and false

  • Symbols - Lightweight constants

  • Strings - Strings of characters

  • Objects - Instantiated objects are defined with the class keyword. Classes are bindings of names to values.

  • Dictionaries - Associative arrays, allocated with the syntax “new {}” or just “{}”

  • Arrays - Arrays, allocated with the syntax “new [X]” where X is an expression computing the size of the array

  • Exceptions - The base type for an error that can be handled in code with a try/catch. Used with the throw keyword.

Instance Method Details

format(format, params) ⇒ String

Use format() to create a formatted String by substituting the given parameters into the given format at the corresponding locations.

The given format:

"Your next meeting is at $1$:$2$ on $3$ $4$ in room $5$."

Paired with the given parameters:

[2, 30, "Sep", 4, "6820"]

Would result in the String:

"Your next meeting is at 2:30 on Sep 4 in room 6820."

Parameters:

  • format (String) —

    A string using $1$, $2$, $3$… as substitution identifiers

  • params (Array) —

    Array of content to substitute in

Returns:

  • (String) —

    A new String with the substituted content
