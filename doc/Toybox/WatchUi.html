Instance Method Details

animate(drawable, property, type, start, stop, period, callback) ⇒ Object

Use animate() to animate a property over a given period of time. The animation starts after the call and runs the length of the specified period. During this time the View's draw callback will be called.

Parameters:

  • drawable (Drawable) —

    Drawable to animate

  • property (Symbol) —

    Symbol of property to animate over time

  • type —

    Toybox.WatchUi.ANIM_TYPE_* value

  • start (Number) —

    Initial value of property

  • stop (Number) —

    Final value of property after animation

  • period (Float) —

    Time in seconds for animation to run

  • callback (Method) —

    Method object to call when complete, or null

switchToView(view, delegate, transition) ⇒ Object

Use switchToView() to pop the current View and push the new one. Native views (e.g. progress bars, confirmation dialogues, number pickers, etc.) may not be passed as an input parameter. switchToView() only supports switching to non-native views.

Parameters:

  • view (View) —

    View object to push

  • delegate —

    the input delegate to handle input for the View

  • transition —

    View transition of type SLIDE_*
