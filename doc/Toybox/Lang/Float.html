Overview

Floats are 32 bit floating point values. A decimal number is Float by default unless the 'd' prefix is used.

#format(format) ⇒ String

Use format() to format the Float using the given formatting String. The format string is similar to the format string that can be passed in to printf in c stdio (the [length] option is not available):

"%[flags][width][.precision]specifier"

The supported specifiers are: d, i, u, o, x, X, f, e, E.

flags

supports only + and 0

width

supports only numbers (* is not supported)

.precision

supports only numbers (* is not supported)

    A formatted String

#toNumber ⇒ Number
