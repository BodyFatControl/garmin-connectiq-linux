Overview

Double represents a 64 bit floating point number. To use a double in Monkey C add 'd' to the end of the number:

var e = 2.718281828459045d

#format(format) ⇒ String

Use format() to format the Double using the given formatting String. The format string is similar to the format string that can be passed in to printf in c stdio (the [length] option is not available):

"%[flags][width][.precision]specifier"

The supported specifiers are: d, i, u, o, x, X, f, e, E.

flags

supports only + and 0

width

supports only numbers (* is not supported)

.precision

supports only numbers (* is not supported)

    A formatted String
