Overview

Deprecated.

The NumberPicker class will be dropped in Connect IQ 3.0.0

This is the on-screen representation of number picker. The look-and-feel will be device specific. NumberPickers are pushed using WatchUi.pushView(), providing a NumberPickerDelegate for the input delegate. There are minimum and maximum values enforced by the product for each mode. The initial value will be modified to be within these bounds.
