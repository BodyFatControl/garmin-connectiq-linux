Overview

Menu is a representation of an on-screen menu. The look-and-feel of the menu will be device specific. It presents the user with a list of options and after the user chooses one, the registered MenuInputDelegate.onMenuItem() will be called. While a Menu can be generated programatically, in general they should be created as a resource. Menus are pushed using WatchUi.pushView(), providing a MenuInputDelegate for the input delegate.
