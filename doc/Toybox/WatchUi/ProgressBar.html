Overview

This is the on-screen representation of a progress bar. The look-and-feel will be device specific. ProgressBars are pushed using WatchUi.pushView(), providing either null or a behavior delegate. If a behavior delegate is provided, the onBack method will be called if the user presses the back button from the progressBar view. This can display a string and either a progress bar from 0-100% or a busy bar.
