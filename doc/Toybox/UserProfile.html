Instance Method Details

#getCurrentSport ⇒ Number

Returns the sport that the HR_ZONE SPORT_XXX enum that the current activity uses heart rate zones from. If the active sport does not have sport specific zones, it will retrun HR_ZONE_SPORT_GENERIC.

Returns:

  • (Number) —

    Returns the current HR zone sport from the HR_ZONE_SPORT_XXX enum.

Since:

  • 1.2.6

#getHeartRateZones(sport) ⇒ Array

Returns Array of zone thresholds for the requested sport. The returned array contains zone values as follows:

min zone 1, max zone 1, max zone 2, max zone 3, max zone 4, max zone 5

Parameters:

  • sport (Number) —

    The sport that zones are being requested from. Should be a HR_ZONE_SPORT_XXX value.

Returns:

  • (Array) —

    Array of zone thresholds for the requested sport. The returned array contains zone values as follows:
