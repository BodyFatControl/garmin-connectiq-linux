MESG_TYPE_SESSION = 18

MESG_TYPE enum maps to the FIT Mesg types that are written to the activity file.

Mesg Type for Session Messages Written once per session at the end of recording Used for data that pertains to the entire session (eg. Avg Speed)

MESG_TYPE_RECORD = 20

Mesg Type for Record Messages Depending on device recording type written once a second or when new data is available (Smart Recording). Never written faster than once a second. Used for instantaneous values (eg. Current Speed)
