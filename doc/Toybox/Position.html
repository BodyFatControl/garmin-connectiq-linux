Request a location event with enableLocationEvents().

Parameters:

  • type —

    LOCATION_ONE_SHOT for a single location request, LOCATION_CONTINUOUS to enable location tracking, and LOCATION_DISABLE to turn off location tracking

  • listener (Method) —

    Method object to call with location updates; receives a Position.Info as an argument
