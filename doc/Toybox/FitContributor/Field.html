setData(input) ⇒ Object

Set data sets the data to be written out for this field. This function must be called every time an update is to be recorded into the activity recording. If this method is called before the a message is dropped into the activity, the previous value specified will be lost. Do not for time sensitive data with sub second granularity.

Raises:

  • UnexpectedTypeException Thrown if input does not match the type specified at definition.
