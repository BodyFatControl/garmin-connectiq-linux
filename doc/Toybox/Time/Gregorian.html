Module: Toybox::Time::Gregorian

    Moment object to get Info from

  • format (Number) —

    Time FORMAT_XXX constant to use

Returns:

  • (Info) —

    For FORMAT_SHORT, all values in Info are integers. For FORMAT_MEDIUM and FORMAT_LONG, the month/day fields have the strings for the date in question. The time returned has been converted to local time.
